57Digital

  • Language
  • Watch
  • Edit

57Digital is a mobile application development company based in Sheffield, England. It is run by Jamie Chapman and Luke Dixon, two software engineers from Sheffield.

57Digital is responsible for creating an official[1] Minecraft companion guide, Minecraft Explorer.

Minecraft Apps Edit

  • Minecraft Explorer Pro
  • Minecraft Skin Studio
  • Minecraft Papercraft Studio
  • Minecraft Skin Scanner
  • Seeds Explorer - Minecraft Edition
