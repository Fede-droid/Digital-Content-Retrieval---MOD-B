< Bedrock Edition version history
This article is about Bedrock Edition development versions. For Java Edition development versions, see Java Edition version history/Development versions.
This is not an official version history or changelog. For the official list, please visit Release Changelogs – Minecraft Feedback.
For the development versions, please visit Beta and Preview Information and Changelogs – Minecraft Feedback.

Starting on November 22, 2013,[1] Mojang Studios began to publicly release testing versions of full updates to Android users who opt into the beta program, in order to get major feedback, especially for bug reporting. This enabled the official updates to be considerably more stable.

Previous versions (from 0.8.0 to v0.12.1) required opting into a Google+ group to receive development builds. Players would then see development updates appear as normal updates in the Play Store. On July 17, 2015, the Google+ group was removed due to the amount of spam and advertisements that the group attracted.[2][3][4] On November 3, 2015, the Pocket Edition Beta team published a dedicated blog[5] which was subsequently used for users to opt into the v0.13.0 beta program and to display changelogs for these development builds.

On July 31, 2017, the beta program expanded to Windows and Xbox One in anticipation of the Better Together Update.[6]

On January 31, 2022, it was announced that the beta program would be expanded to other platforms, starting with iOS and iPadOS, in the form of Minecraft Preview.[7] The current beta program is expected to be phased out through the year and ultimately be replaced by Minecraft Preview.

Since May 25, 2022, the beta program on Windows and Xbox will not receive any further updates, and the existing beta installation will be automatically updated to the retail (non-beta) versions.[8]

Usually, betas and Minecraft Preview versions are released on Wednesday or Thursday.
