Pages waiting for deletion by an administrator.

To put a page in this category, use {{delete|reason}} or its shortcut {{d|reason}}.

If a page is obviously only vandalism, spam, hoax content, or illegal content, use the template {{speedy delete|reason}} or the shortcut {{sd|reason}} and the page will be added to Category:Pending speedy deletion instead.

If there is consensus to delete, but cannot yet be deleted due to technical reasons, use the template {{delete when|criteria|reason}} and the page will be added to Category:Awaiting deletion instead.

Remember there are some alternatives to deletion, such as editing and improving the page, merging, or redirection, all of which can be done by non-administrators.
