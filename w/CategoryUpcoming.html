This category contains:

  • Pages with instances of {{Upcoming}}.
  • Any instances of {{Crafting}}, {{Smelting}}, {{Brewing}}, {{Achievements}}, or {{Infobox command}} with the parameter {{{upcoming=}}} set.

When a page has content that should change once the next Minecraft update is released, add {{Upcoming}} to it. It is there so that the pages can be changed quickly when an update is released.

Please note that {{Upcoming}} is used to mark references to upcoming features, as opposed to pages about upcoming features.
