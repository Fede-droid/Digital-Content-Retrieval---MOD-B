Crafting descriptions

  • Language
  • Watch
  • Edit
This article is about the descriptions shown in Pocket Edition. For their Legacy Console Edition equivalents, see Legacy Console Edition descriptions.
This page describes content that exists only in outdated versions of Minecraft. 
Item descriptions from crafting/stonecutting were removed in Pocket Edition v0.9.0 alpha, and from smelting in a later update.

In alpha builds of Pocket Edition, crafting interfaces held descriptions for the item to be crafted.

List Edit

Below is a table of all item descriptions. Descriptions for uncraftable items also appear to exist despite not having anywhere to be displayed.

All items are listed in the order as they are found in the en_US.lang file, which means blocks and items are not grouped by material or utility. The table is sortable and can be sorted alphabetically.

Notes Edit

  1. ↑ The following text is listed in the line below this one in en_US.lang:
    Can also be used for low-level lighting.
  2. ↑ Powered rails use the desc.goldenrail string below.
  3. ↑ Light gray dye uses the desc.dyepowder.silver string above. The following text is listed in the line below this one in en_US.lang:
    (Note: light gray dye can also be made by combining gray dye with bone meal, letting you make four light gray dyes from every ink sac instead of three.)
  4. ↑ Jack o'lanterns use the desc.litpumpkin string below.
  5. ↑ Melon blocks use the desc.melon string above.
  6. ↑ Jungle wood stairs are listed twice in en_US.lang.
  7. ↑ Cobblestone walls attempted to take their crafting description from desc.tile.cobbleWall, which did not exist.
