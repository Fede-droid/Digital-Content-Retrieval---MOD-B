Event servers, known officially by the name Live Event, were public event servers hosted by Mojang Studios for a limited amount of time on the main menu on Bedrock Edition. Players have to login to a Microsoft/Xbox Live account in order to join the Live Event server[1] and all devices including consoles can join them just like featured servers.

When a Live Event is announced and opened, a dedicated button appears on the main menu. Upon clicking the button, the player is shown a screen with the server key art, a description, a button that redirects to a web page, and a button to join the event server.

Safety Edit

Xbox Live is a social network for gamers, and servers require the player to have an Xbox Live/Microsoft account.[11] All of the event servers have the chat turned off.
