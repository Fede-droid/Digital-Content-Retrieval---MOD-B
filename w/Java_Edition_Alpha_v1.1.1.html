"Seecret Saturday" redirects here. For seecret updates in general, see Seecret Updates.

Alpha v1.1.1,[2] also known as Seecret Saturday,[1] is the tenth and final seecret update, and is also the only Seecret Update released on Saturday, which added sneaking, new paintings and fishing rods. It was released with server 0.2.1 on September 18, 2010, and is the only Seecret Saturday update; all other nine seecret updates were released on Fridays.

This version is notable for including debug code relating to the gamma slider. This would set the display's gamma and contrast values to zero, resulting in the screen going completely gray.[3] This "gray screen" bug would be patched in Alpha v1.1.2.

Due to a large influx of users downloading the game at the same time, it took an incredibly long time to download. About 205 minutes after it was made available, it was replaced with Alpha v1.1.2.

Additions Edit

Items Edit

Fishing rods
  • Have no use.
  • Are stackable.

Non-mob entities Edit

Paintings
  • Added 5 new paintings:
    • "Graham"
    • "Creebet"
    • "Donkey Kong"
    • "Skeleton"
    • "Pigscene"

Gameplay Edit

  • Sneaking

General Edit

  • Added the ability to change mouse sensitivity.
  • Added sliders to change volume of sound and music.

Changes Edit

Non-mob entities Edit

Paintings
  • Two paintings have had their textures changed to their modern versions.
    • Changed Sea painting from to
    • Changed Stage painting from to

General Edit

  • Option to toggle the debug menu in the options has been replaced with the F3 key. F3 + S reloads all web-loaded resources.
  • Smoother day-to-night and night-to-day transitions.

Fixes Edit

2 bugs fixed
  • Infinite snowball exploit fixed where snow blocks (requiring 4 snowballs) would be crafted then broken, producing 4–6 snowballs back.
  • Solid snow blocks now only drop four snowballs.

Trivia Edit

  • Despite this version being available for several hours, considerably longer than other archived versions such as pre-reuploads of Beta 1.1, there were no known archives of it. An archived copy was not found until June 25, 2021, when Twitter user LunaSorcery provided an archived jar file.[4]
  • There was a bug with crouching that caused the outer layer of custom skins render as if the player wasn't crouching.
  • This version is identical to Alpha v1.1.2, with the exception of the gamma slider debug code, which caused the infamous "gray screen bug".
  • Contrary to popular belief, calm4.ogg does not exist in this version's files.
