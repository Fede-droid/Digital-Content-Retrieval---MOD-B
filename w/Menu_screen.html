Menu screen

  • Language
  • Watch
  • Edit

Minecraft features a multitude of menu screens:

  • Title Screen, the menu shown upon game startup allowing for access to worlds
  • Pause menu, the menu accessible during gameplay
  • World selection menu, allowing one of multiple worlds to be accessed
  • Create New World, a menu allowing for the configuration of multiple aspects for a new world
  • Options, which are configurable via a series of menus
  • Controls, a subset of options, also given a dedicated menu
