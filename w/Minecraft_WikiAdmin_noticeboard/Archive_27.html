Hardness information in infobox is gone

As the title says, the hardness information in the infobox for blocks is gone. It was usually always there, but since, as far as I know, it is gone since at least today, right at the moment when I needed it to create a mod... :P What happened to it? —:nl:Gebruiker:DarkShadowTNT NL Admin (:nl:Overleg gebruiker:DarkShadowTNT ♦ :nl:Speciaal:Bijdragen/DarkShadowTNT) 21:11, 26 November 2016 (UTC)

It looks like it was removed back in July after some discussion on the talk page. Not entirely sure I agree with removal of the field, but that's why it was removed. It looks like Hardness redirects to a table with the same values (though, I'm not sure how up-to-date it is).
Though if you're writing a mod, the information is available on blocks themselves (Block.getBlockHardness() exists, though it's deprecated; there's some IBlockProperties method that you can use to get it as well (which should work with a block state)). This isn't the best place to talk about that though. --Pokechu22 (talk) 21:29, 26 November 2016 (UTC)
It was removed because it's already on the page. Look at the table to the left of the infobox on any block page. –Majr ᐸ Talk
