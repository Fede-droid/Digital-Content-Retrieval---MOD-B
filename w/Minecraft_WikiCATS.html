Minecraft Wiki:CATS

  • Language
  • Watch

CATS (CATS Attack The Spam) is a custom global user group across all language versions of the Minecraft Wiki. Members of this group are trusted community members specializing in removing spam and vandalism across all language wikis. To do this, they gain various rights such as the ability to block users and delete pages. Notably, those rights can be only used to assist in removal of spam and vandalism. This group came into effect on November 24, 2023, after internal discussions prompted by spam and vandalism occurring on some language wikis.

User rights Edit

  • Block other users from editing (block)
  • Bypass IP blocks, auto-blocks and range blocks (ipblock-exempt)
  • Change protection settings and edit cascade-protected pages (protect)
  • Create or modify abuse filters (abusefilter-modify)
  • Create or modify global abuse filters (abusefilter-modify-global)
  • Delete and undelete specific log entries (deletelogentry)
  • Delete and undelete specific revisions of pages (deleterevision)
  • Delete pages (delete)
  • Disable global blocks locally (globalblock-whitelist)
  • Edit pages protected as "Allow only administrators" (editprotected)
  • Edit pages protected as "Allow only autoconfirmed users" (editsemiprotected)
  • Edit pages protected as "Allow only directors" (directoreditprotected)
  • Have one's own edits automatically marked as patrolled (autopatrol)
  • Limit actions that can be performed for some groups for a limited time (protectsite)
  • Make and remove global blocks (globalblock)
  • Mark rolled-back edits as bot edits (markbotedits)
  • Modify abuse filters with restricted actions (abusefilter-modify-restricted)
  • Move files (movefile)
  • Not be affected by IP-based rate limits (autoconfirmed)
  • Not be affected by rate limits (noratelimit)
  • Not create redirects from source pages when moving pages (suppressredirect)
  • Override files on the shared media repository locally (reupload-shared)
  • Override the disallowed titles or usernames list (tboverride)
  • Override the spoofing checks (override-antispoof)
  • Overwrite existing files (reupload)
  • Perform CAPTCHA-triggering actions without having to go through the CAPTCHA (skipcaptcha)
  • Propose new OAuth consumers (mwoauthproposeconsumer)
  • Quickly rollback the edits of the last user who edited a particular page (rollback)
  • Reset failed or transcoded videos so they are inserted into the job queue again (transcode-reset)
  • Revert all changes by a given abuse filter (abusefilter-revert)
  • Search deleted pages (browsearchive)
  • Unblock oneself (unblockself)
  • Undelete a page (undelete)
  • Update OAuth consumers you control (mwoauthupdateownconsumer)
  • Use higher limits in API queries (apihighlimits)
  • View deleted history entries, without their associated text (deletedhistory)
  • View deleted text and changes between deleted revisions (deletedtext)
  • View detailed abuse log entries (abusefilter-log-detail)
  • View disallowed title list log (titleblacklistlog)
  • View log entries of abuse filters marked as private (abusefilter-log-private)
  • View the spam block list log (spamblacklistlog)
  • Add group to own account: Bots
  • Remove group from own account: Bots

Contacting CATS Edit

Currently members of CATS can be contacted via Discord server as well as their respective talk pages. In the future there might be more ways to contact members of this group.
