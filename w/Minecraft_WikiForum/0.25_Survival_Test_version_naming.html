0.25 Survival Test version naming

The version 0.25 SURVIVAL TEST was reuploaded three separate times, and then a subsequent version named 0.25_05 SURVIVAL TEST was released, implicitly numbering the prior versions (incorrectly! Even Notch got confused by these!!).

This is a list of each version with its in-game title:

  1. "0.25 SURVIVAL TEST", #1 original
  2. "0.25 SURVIVAL TEST", #2 / reupload 1
  3. "0.25 SURVIVAL TEST", #3 / reupload 2
  4. "0.25 SURVIVAL TEST", #4 / reupload 3
  5. "0.25_05 SURVIVAL TEST", #5

They are currently titled, respectively, "0.25", "0.25_01", "0.25_02", "0.25_03", "0.25_05". This is bad because (A) This is completely conjectural naming and (B) This is CONFUSING! One Two Three ... Five? That doesn't make sense. Only when listed as "#1 #2 #3 #4 #5" does the numbering make sense.

So - what should be done about these page's titles?

Option 1: no change

Outcome:

  1. 0.25 SURVIVAL TEST
  2. 0.25_01 SURVIVAL TEST
  3. 0.25_02 SURVIVAL TEST
  4. 0.25_03 SURVIVAL TEST
  5. 0.25_05 SURVIVAL TEST

Option 2: just append a number to each

Outcome:

  1. 0.25 SURVIVAL TEST
  2. 0.25 SURVIVAL TEST 2
  3. 0.25 SURVIVAL TEST 3
  4. 0.25 SURVIVAL TEST 4
  5. 0.25_05 SURVIVAL TEST

Option 2a: ... including #1

Outcome:

  1. 0.25 SURVIVAL TEST 1
  2. 0.25 SURVIVAL TEST 2
  3. 0.25 SURVIVAL TEST 3
  4. 0.25 SURVIVAL TEST 4
  5. 0.25_05 SURVIVAL TEST

Option 3: bracketed number to show that the numbering is conjectural

Outcome:

  1. 0.25 SURVIVAL TEST
  2. 0.25 SURVIVAL TEST (2)
  3. 0.25 SURVIVAL TEST (3)
  4. 0.25 SURVIVAL TEST (4)
  5. 0.25_05 SURVIVAL TEST

Option 3a: ... including #1

Outcome:

  1. 0.25 SURVIVAL TEST (1)
  2. 0.25 SURVIVAL TEST (2)
  3. 0.25 SURVIVAL TEST (3)
  4. 0.25 SURVIVAL TEST (4)
  5. 0.25_05 SURVIVAL TEST

Option 4: something else?
