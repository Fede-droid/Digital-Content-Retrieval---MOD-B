Poisonous Potato Add-On

Should we document the Poisonous Potato Add-On for bedrock edition the same way we document the April Fools snapshots for java? I think personaly we should but it might be a slippery slope.

Discussion Edit

 Support as proposer. --Gtbot2007 (talk) 23:52, 25 May 2024 (UTC)Reply[reply]
We recently decided to document official add-ons and other Marketplace content on the wiki (see the discussion here), so we can make a page for the poisonous potato add on. I’ve been meaning to start working on the page, but if you want to get started with it then that would be awesome. - Harristic / Talk 23:55, 25 May 2024 (UTC)Reply[reply]
The difference is that most marketplace content is a just a self contained map, while this is addon that could be used in any world, so what I was thinking is multiple pages like we did for the April Fools snapshots Gtbot2007 (talk) 00:02, 26 May 2024 (UTC)Reply[reply]
I'm not too sure how that would work. I think we should try to aim for a single page, quite honestly, since multiple pages might get confusing -- ThatOneWolf Time for a chat?See my edits? 11:11, 26 May 2024 (UTC)Reply[reply]
For now we've decided to just have all information about an add-on or other Marketplace content on one page. Marketplace content is still second-party content, whereas joke snapshots are wholly first-party. Maybe there could be a further discussion about this but I personally wouldn't support it, it just feels like too many pages about things that people don't care about as much, that can be easily documented on one page. Features in 24w14potato for example are a lot more expansive, too much to document on one page. - Harristic / Talk 11:16, 26 May 2024 (UTC)Reply[reply]
