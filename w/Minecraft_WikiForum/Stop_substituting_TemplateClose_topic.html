Stop substituting Template:Close topic

It's regarded as best practice to subst talk page template so that people's comments are not modified when the template is updated. However, that does not seem to be an issue with {{Close topic}}. In addition, we've had to run bot multiple times to fix issues with this template, most recently to fix dark mode issues. Now that DiscussionTools is installed, we have to run a bot again to replace all Close topics in the wild to add a class to the box so that DiscussionTools no longer displays a reply button inside it. I don't see any real benefit in substing this template. I suggest we stop doing it, and we replace Close topics using a bot for the last time to un-subst all usages.
