Style guide for commands

Proposal Edit

This is a proposal for a style guide for commands and command articles.

Reasoning Edit

Command articles are the most popular technical articles on this wiki, but they are difficult to write. Every time a new command was added into the game, its article stayed in a messy state for a long time. A comprehensive style guide is necessary to guide editors in writing them.

Currently, there is a draft for command style guide HERE. Because in recent years, most command pages have been rewritten by me, most command pages have already followed this style guide draft. Feel free to discuss or leave your suggestions below, or edit the draft. The style guide should be applied if there are no issues.

Thanks.--Chixvv (talk) 17:21, 21 February 2024 (UTC)Reply[reply]
