What to do with "lost version" message boxes

Currently, pages like Java Edition 1.6.2 feature message boxes explaining how the version is missing using {{Lost version}} and the like. This is how it appears:

The pre-reupload version of this version is currently missing. 
This version has been reuploaded at one point in time. While the reuploaded version has been archived, the pre-reupload has not, meaning that it is currently lost.
If you believe you have a copy of this version, please post it on this page's talk page.
The pre-release version of this version is currently missing. 
This version had a pre-release at one point in time. While the release version has been archived, the pre-release has not, meaning that it is currently lost.
If you believe you have a copy of this version, please post it on this page's talk page.
1.6.2
Release date

etc 

{
    "title": "1.6.2",
    "rows": [
        {
            "field": "etc",
            "label": "Release date"
        }
    ],
    "invimages": [],
    "images": []
}

1.6.2 is a minor update to Minecraft (Java Edition) released on July 8, 2013.

That is a lot of text-wall information to put at the very top of the article where the viewer starts reading from. This is a problem, in my eyes.

There are two solutions to this:

 1. Put the message boxes smaller, off to the side

1.6.2
Release date

etc 

{
    "title": "1.6.2",
    "rows": [
        {
            "field": "etc",
            "label": "Release date"
        }
    ],
    "invimages": [],
    "images": []
}
The pre-reupload version of this version is currently missing. 
This version has been reuploaded at one point in time. While the reuploaded version has been archived, the pre-reupload has not, meaning that it is currently lost.
If you believe you have a copy of this version, please post it on this page's talk page.
The pre-release version of this version is currently missing. 
This version had a pre-release at one point in time. While the release version has been archived, the pre-release has not, meaning that it is currently lost.
If you believe you have a copy of this version, please post it on this page's talk page.

1.6.2 is a minor update to Minecraft (Java Edition) released on July 8, 2013.

 2. Merge all the information in the message boxes into one template

This version is currently missing it's pre-reupload and pre-release versions. 
These versions are not available in the launcher and are therefore currently lost.
If you believe you have a copy of this version, please post it on this page's talk page.
1.6.2
Release date

etc 

{
    "title": "1.6.2",
    "rows": [
        {
            "field": "etc",
            "label": "Release date"
        }
    ],
    "invimages": [],
    "images": []
}

1.6.2 is a minor update to Minecraft (Java Edition) released on July 8, 2013.


 3. Or: Both (probably not)

1.6.2
Release date

etc 

{
    "title": "1.6.2",
    "rows": [
        {
            "field": "etc",
            "label": "Release date"
        }
    ],
    "invimages": [],
    "images": []
}
This version is currently missing it's pre-reupload and pre-release versions. 
These versions are not available in the launcher and are therefore currently lost.
If you believe you have a copy of this version, please post it on this page's talk page.

1.6.2 is a minor update to Minecraft (Java Edition) released on July 8, 2013.
