Workstations overhaul

I am working on a new page, called "Workstation". You can find it at User:Simanelix/Workstation.

The proposal is as follows:

  • Move User:Simanelix/Workstation
    • On crafting table, we should add {{Redirect|Workbench|other blocks work-related blocks|Workstation}}, because "workstation" and "workbench" could be confused with eachother.
  • Change the desscriptions of all blocks listed in User:Simanelix/Workstation to say "is/are a workstation".
  • Create a page for each of the (workstation) prcoesses listed in User:Simanelix/Workstation.
    • Process would be too easy to confuse with things like the game's program process, so a name like Workstation process would be better. This might be better of as a section, i.e. Workstation § Process.
    • Describe these process as "workstation processes", both on their page, and on the pages of blocks whose usage involves that process.
      • Describe enchanting as "a workstation process using the enchanting table".
      • Describe anvil mechanics as "workstation processes using the anvil".
      • Describe item repair as "a workstation process using the anvil, smithing table, or crafting grid".
      • Describe crafting as "a workstation process using the crafting grid". Then describe "A crafting grid is a grid of inventory slots. The player has a 2x2 crafting grid in their inventory, and the crafting table has a 3x3 crafting grid in its GUI. The inventory slots in a crafting grid can only store items while the GUI is open. If a player closes the GUI, the items will try to move into the player's inventory slots (not the slots in the crafting grid in the player's inventory), and will spew out of the player if there is no room in the inventory."
  • Add GUI sub-sections to the Usage sections of various block and workstation process pages.

I know it seems like I am proposing a lot of things here. That's because this is a big change and would require a lot of work.
