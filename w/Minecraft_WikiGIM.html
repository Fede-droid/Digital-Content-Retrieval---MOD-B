Global interface maintainers are trusted community members specializing in improving and implementing fixes to user interface, especially CSS and JS across all language wikis. To do this, they gain the rights to edit these pages and all protected pages across all language versions of the Minecraft Wiki. This group came into effect on November 24, 2023, after internal discussions on helping to implementing fixes across all language wikis.

User rights Edit

  • Change protection settings and edit cascade-protected pages (protect)
  • Edit other users' CSS files (editusercss)
  • Edit other users' JSON files (edituserjson)
  • Edit other users' JavaScript files (edituserjs)
  • Edit pages protected as "Allow only administrators" (editprotected)
  • Edit pages protected as "Allow only autoconfirmed users" (editsemiprotected)
  • Edit pages protected as "Allow only directors" (directoreditprotected)
  • Edit sitewide CSS (editsitecss)
  • Edit sitewide JSON (editsitejson)
  • Edit sitewide JavaScript (editsitejs)
  • Edit the content model of a page (editcontentmodel)
  • Edit the user interface (editinterface)

Contacting global interface maintainers Edit

Currently global interface maintainers can be contacted via Discord server as well as their respective talk pages.
