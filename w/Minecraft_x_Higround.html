Minecraft x Higround

  • Language
  • Watch
  • Edit

Minecraft x Higround is a collaboration between Minecraft and gaming keyboard company Higround.[1] Various keyboards, mousepads, and other merchandise are set to be released as part of the collaboration to celebrate Minecraft's 15th anniversary. The merchandise is set to have a limited release on May 29, 2024, at 19:00 UTC.

Six keyboards are set to release as part of the collaboration.
