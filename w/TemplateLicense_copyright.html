This template should be used on any files that are under copyright.

Usage

This template can be automatically added to a file on upload by selecting This file is copyrighted, but use is permitted by the copyright holder from the licensing drop-down menu.

Alternatively, this template can be manually added to a file by pasting this code under the file's summary (if it has one) and above any categories.

If permission has been explicitly granted by the copyright holder, and a link to the tweet/etc using |1=. 

== Licensing ==

Files using this template are automatically added to the Copyrighted images category.
