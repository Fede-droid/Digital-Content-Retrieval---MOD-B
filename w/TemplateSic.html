[sic]

[view] [edit] [history] [refresh]Documentation

This template is for situations where some text being quoted has incorrect/unexpected spelling or grammar, yet it would be preferable to preserve the text exactly, rather than "correct" it.

Usage Edit

This template can simply be added after the word(s) in question.

This template links to Wikipedia:Sic, but can be disabled with |nolink=.

Examples Edit

"Collect a hundred glostone{{sic}} from the Nether." → "Collect a hundred glostone [sic] from the Nether."
