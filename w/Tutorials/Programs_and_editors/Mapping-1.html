This article is about tools that map a world's terrain. For tools that deal with obfuscation maps, see Programs and editors/Modding.

2D map viewers Edit

2D map viewers create a visual map of your world's terrain. Many mappers have options for different overhead and angled modes, and some limit the map to selected block types or slice the map into smaller pieces.


Name Description Platforms Link Status Latest Compatible Release Source available Programming language Author

Minutor

Interactive GUI, shows top-down view of your map with a depth slider to reveal caves. Show mobs, items and other entities. Visualize light levels, where hostile mobs may spawn, biomes, slime chunks and much more. Search for entities and blocks. Overlay for any generated structure.

Forum 

Website 

 Active Java Edition 1.20 Yes, BSD C++ mrkite77, Mc_Etlam

uNmINeD

2D overhead map for Minecraft JE/BE worlds. Underground, x-ray, nether, caves, night view, save to image and HTML. Easy to use GUI and a command line tool.

Forum 

Website 

 Active Java Edition 1.20.1, Bedrock Edition 1.20.10 Not yet C# megasys

Mineways

Mineways is meant more for exporting data into Blender and other digital content creation apps, but is also useful for mapping. It includes the ability to peel away terrain and look below. Similar to Minutor, which it is branched from. (works on Linux/Mac with Wine)

Forum 

Website 

 Active Java Edition 1.20.1 Yes, BSD C, C++ Eric Haines

Little a Map

Little a Map renders a composite of player-created map items with the goal of minimizing external effects on survival gameplay.

Website 

 In development Java Edition 1.20.6 Yes, GPL Rust Andrew Kvalheim

Papyri

Papyri is a Minecraft map item web presenter. It shows all maps created on a server positioned and scaled properly, creating a mosaic of your world as explored with maps.

Website 

 Active Java Edition 1.17 Yes, MIT python greener_ca

PixelMap

Top-down mapper useful mainly to generate huge image renders both fast and with as little resources as possible. GUI and CLI.

Forum 

 In Development Java Edition 1.20, Bedrock Edition 1.20 Not yet C++ McTwist

JourneyMap

Forge Client/Server mod for real-time mapping in SP and MP. Minimap/full-screen or use a web browser. Day/night/surface/cave mapping and waypoints. Bukkit/Spigot server support for multi-world. JourneyMap API lets other mods write plugins to add overlays to maps.

Forum 

Website 

 Active Java Edition 1.12.2 No Java, DHTML Techbrew

Cartographer for Minecraft

A mobile app that allows users to add pins to their favourite places on their own Minecraft Worlds, with biome preview.

Website 

 Active Java Edition 1.20, any Minecraft No Flutter, Dart emmekappa

Bedrock Viz

Bedrock Viz is a fork of MCPE Viz to fix bugs and add new features. It uses Bedrock Edition world files, creating overview images from the world files, reporting many details of the world. It is a web app that can be run locally, but is best served by a server to access all the features.

Website 

 Active Bedrock 1.18.12 Yes, GPL C Jasper Wan

BlockMap

BlockMap is a standalone application that renders beautiful top-down views of your worlds. It also displays points of interest like players and structures as interactive pins on the map. A CLI and library for more custom usage are available as well.

Website 

 Inactive Java Edition 1.18 Yes, MIT Java piegames

c10t

Command-line. A rewrite of Cartograph. Official support for Linux builds, works for Windows and Mac, but requires compilation.

Forum 

Website 

 Inactive Java Edition 1.3.2 if compiled from source on github Yes, BSD C++ udoprog (John-John Tedro)

Ink & Parchment Map

A web-based shareable map where players add locations manually, so no access to the server files is required for players to create a map of the world and share it. Designed to not completely remove the element of exploration from the world. Also works for the Nether. 🌐

Forum 

Website 

 Inactive Java Edition 1.13.1, any Minecraft Yes, GPL HTML5, javascript DrFrankenstone

MapRend

MapRend renders a top down map of Worlds saved in the Minecraft Anvil format. It generates the used colorset from a resource pack, can render day and night maps, shows the height and light of blocks, is easy to customize and easy to update (as long as worlds are saved as MCAnvil)! Can also save smaller images of chunkclusters (e.g. for dynamic loading maps).

Website 

 Inactive Java Edition 1.8.1 and every newer one using MCAnvil Yes, GPL Java YSelf Tool

MCPE Viz

MCPE Viz is a tool that parses Minecraft Pocket Edition (MCPE) world files. It creates overview images from the world files, and reports lots of details on your world.

Website 

 Inactive Bedrock 1.10 Yes, GPL C Plethora777

papyrus.cs

papyrus.cs renders maps of Minecraft: Bedrock Edition worlds using C#, LevelDB and leaflet.

Website 

 Inactive Bedrock Edition 1.16.0 Yes, Apache C# DeepBlue4200

papyrus.js

papyrus.js renders maps of Minecraft: Bedrock Edition worlds using node.js, LevelDB and leaflet.

Website 

 Inactive Bedrock Edition 1.14.0 Yes, Apache JavaScript clarkx86

TOGoS's Minecraft Map Renderer

Simple and fast region-based overhead map imaging tool designed to efficiently update extremely large maps as a set of tiles, one per region, which can be combined by TMCMR or using external tools. Supports custom color mappings and arbitrary translucency for any block.

Website 

 Inactive Java Edition 1.12.2 Yes Java TOGoS

3D map viewers Edit

As the name suggests, 3D map viewers allow you to view your worlds in 3D.

Name Description Platforms Link Status Latest Compatible Release Source available Programming language Author

Chunky

3D path-tracing renderer for Minecraft worlds. Produces very high quality results, but can take a long time to render (best to have it done during the night). Supports worlds from Minecraft 1.2.1 up to 1.20.6 (Anvil file format) and CubicChunks worlds from Minecraft 1.10 up to 1.12.2. Example renders can be found here.

Forum 

Website 

 Active Java Edition 1.20.6 Yes, GPL Java nrub (llbit), leMaik, and the GitHub community

Minecraft Overviewer Successor

A LeafletJS powered overview that draws your map in 30 degree oblique mode with a high level of detail. An example render is available here.

Website 

 Active Java Edition 1.20.4 Yes, GPL Python/C Overviewer Team, GregoryAM, Simon Walker, and others.

mcmap

Command-line. mcmap is an optimized isometric pixel-art map viewer supporting the latest versions of Minecraft. Development is ongoing and feature requests are appreciated. Currently built from source. Support for Windows via Ubuntu for Windows.

Website 

 In development Java Edition 1.20.1 Yes, GPL C++ Zahl / WRIM / spoutn1k

BlueMap

Each chunk is transferred and rendered in the browser, leading to a 3D view over the map as well as an overview instead of the most common pre-rendered on the server. Compatible with Spigot/Paper, Sponge, Fabric and Forge, as well as running stand-alone.

Website 

 Active Java Edition 1.20.2 Yes, MIT Java/HTML/CSS BlueMap

Dynmap

A Google Maps like interface on Minecraft, accessible through a web browser. Compatible with Bukkit and Forge, and has an API to render of custom elements.

Forum 

Website 

 Active Java Edition 1.8.9 - Java Edition 1.20.1 Yes, BSD Java/HTML/CSS mikeprimm

Avoyd

Avoyd is a voxel editor, viewer and renderer with the ability to load the largest Minecraft maps and display and explore them in their entirety. World size can go up to 260k x 260k x 260k voxels. Light and atmosphere can be modified. Cameras: first person and arcball cameras, teleport and save multiple camera positions. Imports Minecraft Maps, .vox files and multi-models from MagicaVoxel. Exports to .obj (Blender 3D). High quality renders with CPU / GPU path tracing and denoising.

Forum 

Website 

 Active Java Edition 1.20.6 No C++ enkisoftware

Tectonicus

Renders high-detail zoomable versions of minecraft maps, viewable via Leaflet. Command line only.

View example map

Website 

 Active Java Edition 1.16.3 Yes, BSD Java OrangyTang and Tectonicus contributors

Minecraft Overviewer

A LeafletJS powered overview that draws your map in 30 degree oblique mode with a high level of detail.

Website 

 Inactive Java Edition 1.19.4 Yes, GPL Python/C Overviewer Team

Minecraft X-Ray

3D map viewer that highlights minerals. Loads Minecraft levels as far back as Alpha, all the way through the current Anvil format. Supports user-supplied custom blocks for mod support.

Forum 

Website 

 Inactive Java Edition 1.5.1 Yes, BSD Java plusminus, xolotl

Pigmap

Fast renderer designed for large worlds. Supported platforms are Linux and Cygwin. No binaries; must be built from source.

Website 

 Inactive Java Edition 1.8 Yes, GPL C++ equalpants + github community contributors

Map editors Edit

Map editors alter an existing level's terrain or other properties, such as player position and time of day. Some of them also let you edit your inventory and generate terrain.

Name Description Platforms Link Status Latest Compatible Release Source available Programming language Author

MCA selector

Fast loading of worlds. Can be used to select (with many different filters), edit, delete, export or import chunks. Can also globally replace block types. Can open multiple instances of the program to load separate worlds to copy from one and paste into another. Can "import selected chunks" and point to a older/newer version of the same world.

Website 

 Active Java Edition Beta 1.3 through 1.20+, and most snapshots Yes, MIT Java Querz

Amulet Editor

Amulet Editor is a versatile world editor and converter written in Python.

Note: The project does not have a license.

Forum 

Website 

 Active Java Edition 1.12+ Bedrock 1.7+ Yes; however, the project does not have a license Python Podshot, naor2013, gentlegiantJGC, BluCode

Creative Mode Plus

Top-down builder for constructing buildings and dungeons quickly and easily for Minecraft maps.

Forum 

Website 

 Inactive Java Edition 1.4.7 Yes C# MyPace (Caleb Waggoner)

MCEdit

The first 3D editor. Created to import older levels into the new Alpha levels. Fly around in 3D and select blocks, then copy or export them. Import blocks from schematics or from entire Classic levels. Resize a level by creating or deleting chunks.

Forum 

Website 

 Inactive Java Edition 1.11 (stable)

Overhead inventory editor and map viewer made for Minecraft Pocket edition, includes rendering of layers and saving renders to png.

Map generators Edit

These programs either create a new level with custom terrain, or "implant" new blocks into an existing level.

Name Description Platforms Link Status Latest Compatible Release Source available Programming language Author

WorldPainter

WorldPainter is an interactive, graphical map generator which allows you to mould a landscape with paint-like tools that can change the shape and type of the terrain, set the biomes, add forests, snow, ice, resources and caverns and much more. It has limited support for editing existing maps, but is primarily a map generator.

Forum 

Website 

 Active Java Edition 1.20+ Yes, GPL Java Captain_Chaos

WorldSynth

WorldSynth is a procedural 3D map generator, with a node-based GUI similar to that of WorldMachine.

Forum 

Website 

 Active Java Edition 1.20.2 Yes, Mozilla Public Licence 2.0 Java booleanbyte

Chunkmapper

Generates the real world in Minecraft at 30:1.

Website 

 Inactive Java Edition 1.7.5 No Java whamtet

MCDungeon

Creates randomly generated dungeons and treasure hunts in an existing level. Can work with Minecraft 1.7, however some 1.8-specific features need to be disabled

Website 

 Inactive Java Edition 1.8 Yes, MIT Python Orphu

Terrain Control

Some of Terrain Controls Features:

- Adjust the ratio of land to ocean and the size of continents - Control global and per-biome terrain parameters - Tweak settings for Minecraft's built-in biomes and your own custom biomes - Fine-tune resource distribution to your taste - Add custom objects and trees to your world - Enable or disable Minecraft's villages, strongholds, pyramids and abandoned mines - Has an User Interface (by Genae) to easily modify the config files - Revert to Minecraft 1.7's terrain generator to continue expanding your legacy worlds - Generate a Map from an Image - Runs with multiplayer servers and on singleplayer.

Forum 

Website 

 Inactive Java Edition 1.7.9 Yes Java Khoorn

TOGoS's Minecraft Map Generator

Generates new chunks based on user-defined functions. Functions have complete control over block placement and can use 3D Perlin and Simplex noise to generate any combination of oceans, continents, caves, lava pits, mountains of wool, giant obsidian sponges, etc.

Forum 

Website 

 Inactive Java Edition 1.5 Yes Java TOGoS

Seibai's Dungeon Adventure Generator

Generates a huge world made of 4 sections. The player goes through dungeons to reach the final world. The dungeons are randomly chosen from a variety of different area-types. The traveling works through nether portals; the player goes into a nether portal, walks along a small hallway and goes into another portal to reach the next area.

Forum 

 Inactive Java Edition 1.2.5 No N/A Seibai

Seibai's Planetoids Map Generator

Generates planetoids made of different materials. The entire map is surrounded by bedrock walls. The stone planetoids keep materials inside such as ores. The leaf-planetoids hold a massive wooden core. NOTE: The Better Worlds Generation mod has added this abandoned mod to it's world types.

Forum 

 Inactive Java Edition 1.2.5 No N/A Seibai

Seibai's Primordial Desert Map Generator

Generates a desert world with dunes, sharp stone mountains and underground dungeons. The map has a bedrock wall around it. Outside of that bedrock wall lies normal terrain. Survival is hard in the desert because food and wood are rare. This still works with 1.5.2, even though it does not generate new ores!

Forum 

 Inactive Java Edition 1.2.5 No N/A Seibai

Seed mappers Edit

Seed viewers replicate Minecraft's biome generation for given seeds, and display the generated biomes in an 2D map. Often, they also display locations of structures in the map.

Name Description Platforms Link Status Latest Compatible Release Source available Programming language Author

Cubiomes Viewer

High performance biome and structure mapper and seed finder utility powered by cubiomes.

Website 

 Active Java Edition 1.1 - Java Edition 1.20.6 Yes, GPL C++ Cubitect and the GitHub community

Seeder

Seeder is a website where you can show and find seeds without ever leaving your browser. Works on mobile. Powered by cubiomes.

Forum 

Website 

 Active Java Edition 1.20 Yes, MIT Javascript (React), C TrinTragula

mcseedmap.net

A browser based seed map viewer and seed finder powered by cubiomes. 🌐

Forum 

Website 

 Active Java Edition 1.20.1, Bedrock Edition 1.20.10 No C/C++/JS megasys

Minemap

Using only a world seed, Minemap deterministically generates a map of the world’s terrain, biomes, and structures. Compared to Amidst, Minemap has expanded features, better performance, and doesn’t require a Minecraft installation.

Website 

 Active Java Edition 1.17 Yes, MIT Java Neil

Amidst

Amidst is a tool to display an overview of a Minecraft world, without actually creating it. Shows biomes and generated structures (e.g. villages). Taps into selected Minecraft binary for generator settings.

Forum 

Website 

 Inactive Java Edition 1.17 Yes, GPL Java Skidoodle, DrFrankenstone and the GitHub community

MineAtlas

A browser-based mapper similar to AMIDST. Denotes points of interest accurately in post 1.8 generated chunks (monuments, temples, etc.,), and can extrapolate potential ones from the seed (slime chunks, villages). Displays biome names on mouse over, though topography is approximated. Requires either the Level.dat file, manual input of seed to function, or it can generate a random seed for examination. Good tool for server owners, and renders huge, zoomable, dragable maps is seconds, displaying even unexplored areas quickly. Maps are by necessity low resolution, and max zoom is not high, wlthough it can display well over 50 km x 50 km sections of map on the farthest zoom level. 🌐

Forum 

Website 

 Inactive Java Edition 1.8+ May support older builds in future. No ? jazzyjaffa

Obscure software Edit

This section lists software that has been discontinued or left unmaintained for years, and for which downloads, websites, etc. are difficult to find, or no longer work, etc. One may need to search hard and in the Wayback Machine to find working and safe downloads. Unused domains can be repurposed, sometimes for malicious purposes, so exercise caution when searching so as not to fall victim to scams and malware.


Name Description Platforms Link Status Latest Compatible Release Source available Programming language Author

Blocktopograph

Free top-down world viewer for Minecraft Pocket Edition (MCPE), includes NBT editor as well. Still in "Beta", stable but not feature complete. Current features: Map-types: overworld, cave, heightmap, biome, grass color, X-Ray and block-light. With markers and coordinates. Local player can be teleported.

Website 

 Inactive Pocket Edition 1.0 No Java ProtoLambda

Cartograph G

Branch from Cartograph. Supports the latest updates and some fixes of Cartograph bugs. New features are added, including an enhanced GUI and support for mods. Current compatibility relies on user generated Auto.cfg files. In danger of becoming abandon-ware

Forum 

 Inactive Java Edition 1.4.2 No, but has been cracked C++ (renderer), VB (interface) Gameslinder

Mapcrafter

High Performance Minecraft Map Renderer - Renders Minecraft worlds to high detail maps with isometric 3D and/or 2D topdown perspectives viewable in any webbrowser. Four rendering rotations, day/night/cave render modes, world cropping and many more features available. **Known glitches with this program include incorrect assignments for leaves - needs an update**

Minecraft AutoMap

Displays a highly customizable live-updated map of your surroundings, including nearby creatures and players. Works in Singleplayer and Multiplayer. The map runs in a separate window from Minecraft so it can be sized and positioned, fullscreened, or placed on top of Minecraft like a minimap. Requires Risugami's ModLoader, .NET 4.0, and XNA 4.0.

Forum 

Website 

 Inactive Java Edition 1.7.1 Yes, MS-PL C# bp2008, Ceii

Minecraft Overworld Viewer

Minecraft Overworld Viewer is a tool that can generate a region map (image of 512 x 512 pixels), for each region file (.mca) from a selected Minecraft dimension and then join all of them into a single full world map. It includes 5 map types (default, no liquids, no plants, structures and underground caves). And for massive worlds it also auto generates several 8.192 x 8.192 images with all the excluded regions if they couldn't fit into the big final image, but it's also smart because it first analyzes each exported region when joining and start the full world map at the zone with more blocks on it (or existing chunks), meaning it always tries to include the zones with more bases, etc in the final map.

Forum 

Website 

 Inactive Java Edition 1.18 and newer Yes, GPL C# Jupisoft (Júpiter Mauro)

Minecraft Tools

Minecraft Tools is a multi-tool with almost 100 unique tools in one. And several of them are related to display different views of the Minecraft worlds, like the Overworld viewer or Biomes viewer (works like Amidst). It can do almost all imaginable things and is updated continuously.
