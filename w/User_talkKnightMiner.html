Welcome to my talk page! Before asking a question here, see if your question fits any of the following: 
  • Responding to a topic of mine from another talk page? - Respond on that page first, I usually watch pages on which I create topics.
  • Do you have a problem with one of my templates? - Ask on its talk page.
  • Have a task requiring administrative rights? Post on the admin noticeboard.
  • Need in game help? - Try the support forums or the bug tracker.

Otherwise feel free to ask the question here using the "Add topic" button. Just remember to sign your post using ~~~~.

Including a template css subpage into the template Edit

Latest comment: 29 March 20212 comments2 people in discussion

Hi, thank you for doing this content model change. I had been puzzling over why it wasn't appearing properly.

Perhaps you can advise me on how this stylesheet might be included in its parent template Template:Tree list. I ported it over from Wikipedia, and it mostly works except for the Template:Tree list/branching sub-template, which requires the style sheet.

On Wikipedia this is done with <templatestyles src="Tree list/styles.css" /> but that tag isn't supported here. Any advice? Amatulic (talk) 19:19, 28 March 2021 (UTC)Reply[reply]

Look at how we do the stylesheets for Module:Sprite instead. The key details are it requires the CSS page to be loaded from the MediaWiki namespace (CSS can be a security issue), and it is based on the Mediawiki gadget system. For the record, we already have treeview CSS, any reason your template does not just use the existing CSS? KnightMiner (t/c) 00:38, 29 March 2021 (UTC)Reply[reply]
