Talk page replies should not be marked as minor edits Edit

Latest comment: 27 November 20232 comments2 people in discussion

Hello, GXDdcZ123,

One thing of note with the talk page discussions is to not mark any replies being made on talk pages as minor. When new replies to discussions are marked as minor, it causes issues with people not getting notifications for talk pages, and missing new updates on the discussion.

It is fine if it's an issue such as fixing minor formatting mistakes, and then it's OK to mark it as a minor edit, but otherwise, in most instances, it shouldn't.

Thanks for the understanding. Delvin4519 (talk) 15:09, 27 November 2023 (UTC)Reply[reply]
