This feature is exclusive to Bedrock Edition. 

Virtual reality (VR) is a mode natively exclusive to Bedrock Edition for Windows (formerly Windows 10 Edition) and PlayStation 4. It can be used with the Oculus Rift, Meta Quest (via Link and Air Link), PlayStation VR, and all Windows Mixed Reality compatible headsets. It was previously available on Gear VR until October 2020, when it was discontinued along with some older devices.[1]

Launching[2] Edit

To set up a custom shortcut right click your desktop, under ‘new’ create a new shortcut

For the location of the item, enter the following: minecraft://Mode/?OpenXR=true

Click ‘Next’ and give your shortcut a name like “Minecraft VR”

You should see the shortcut created on your desktop and it should automatically have the Minecraft icon if set up correctly

Click ‘Finish’ and your Minecraft: Bedrock Edition VR shortcut has been created!

Opening Edit

To open Minecraft Bedrock in VR, make sure that your VR goggles are connected to your computer and that they have been turned on.

Open the Minecraft VR shortcut and equip the goggles.

You should see Minecraft open up in VR!
