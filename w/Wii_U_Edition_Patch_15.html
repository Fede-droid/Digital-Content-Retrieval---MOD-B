Patch 15 was a version of Wii U Edition released on December 22, 2016.[1] This version is equivalent to Java Edition 1.10. It is also known as the Holiday Update.[2] [edit]


Additions Edit

General Edit

  • Added new tutorial world.
  • Added Fallout Mashup Pack trial.

Blocks Edit

Chorus flower, chorus plant, dragon head, end rod, purpur block, purpur pillar, purpur stairs, purpur slab.

Items Edit

Chorus fruit, popped chorus fruit, dragon's breath, elytra, end crystal, lingering potion, water splash potion and lingering potion, potion of luck, tipped arrow, boats made from spruce, birch, jungle, acacia and dark oak wood.

Mobs Edit

Shulker, stray, husk and skeleton trap horses

Structures Edit

End gateway, chorus tree

Changes Edit

Blocks Edit

  • Brewing now requires blaze powder as fuel for the brewing stand.

Non-mob entities Edit

  • Boats are now controlled differently and can have multiple passengers.

World generation Edit

  • The End has changed when generating a new world and can also be updated for old worlds. To get the new End in an old world, select "Reset End" in the "More Options" menu when loading the world.
  • After killing the ender dragon in the new End, an end gateway will spawn nearby. Throwing an ender pearl into this will teleport the player to a new area in the End with an end city and end ship.
  • Within the End City and End Ship the player can find shulkers, elytra and a dragon head.
  • Villages now generate naturally in the Taiga biome.

General Edit

  • Updated crafting interface to allow crafting firework stars and fireworks.
  • Reordered potion effects in the Brewing tab in the Creative inventory to be more like Java Edition.
  • Removed all five types of extended-enhanced potion variants for five potion effects from the Creative inventory.
  • Removed the strength indicator from the Brewing tab in the Creative inventory.
