TU1 was a version of Xbox 360 Edition released on May 9, 2012.[1] This version is based on Java Edition Beta 1.6.6. This was the day-one version for consoles.

Changes Edit

  • Updated crafting icons.
  • Texture changes.
  • New clearer font.
  • Save indicator will now disappear after 9 seconds.
  • Changed the chat message background.

Trivia Edit

  • There is a house in the panorama screen which isn't found on the tutorial world.
